## Installation Paths

When cloning, the repository will be named `bucucore-framework` but should be renamed to `bucu-core` for FiveM compatibility:

```bash
# Clone with automatic rename
git clone https://github.com/bucucore-dev/bucucore-framework.git bucu-core

# Or clone and rename manually
git clone https://github.com/bucucore-dev/bucucore-framework.git
mv bucucore-framework bucu-core
```

## NPM Package (CLI Tool)

The CLI tool will be published as:
- **Package Name:** `@bucucore/cli` or `bucu-cli`
- **Install:** `npm install -g bucu-cli`
